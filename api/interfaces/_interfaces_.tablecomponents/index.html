Interface: TableComponents<Context>

"interfaces".TableComponents

Customize the TableVirtuoso rendering by passing a set of custom components.

Type parameters#

NameDefault
Contextunknown

Hierarchy#

  • TableComponents

Properties#

EmptyPlaceholder#

• Optional EmptyPlaceholder: ComponentType<{ context?: Context }>

Defined in src/interfaces.ts:177

Set to render a custom UI when the list is empty.

FillerRow#

• Optional FillerRow: ComponentType<FillerRowProps & { context?: Context }>

Defined in src/interfaces.ts:187

Set to render an empty item placeholder.

ScrollSeekPlaceholder#

• Optional ScrollSeekPlaceholder: ComponentType<ScrollSeekPlaceholderProps & { context?: Context }>

Defined in src/interfaces.ts:182

Set to render an item placeholder when the user scrolls fast. See the scrollSeek property for more details.

Scroller#

• Optional Scroller: ComponentType<ScrollerProps & { context?: Context }>

Defined in src/interfaces.ts:167

Set to customize the outermost scrollable element. This should not be necessary in general, as the component passes its HTML attribute props to it.

Table#

• Optional Table: ComponentType<TableProps & { context?: Context }>

Defined in src/interfaces.ts:145

Set to customize the wrapping table element.

TableBody#

• Optional TableBody: ComponentType<TableBodyProps & { context?: Context }>

Defined in src/interfaces.ts:172

Set to customize the items wrapper. Default is tbody.

TableFoot#

• Optional TableFoot: ComponentType<Pick<ComponentPropsWithRef<"tfoot">, "style" | "ref"> & { context?: Context }>

Defined in src/interfaces.ts:156

Set to render a fixed footer at the bottom of the table (tfoot). use [[fixedFooterContent]] to set the contents

TableHead#

• Optional TableHead: ComponentType<Pick<ComponentPropsWithRef<"thead">, "style" | "ref"> & { context?: Context }>

Defined in src/interfaces.ts:151

Set to render a fixed header at the top of the table (thead). use [[fixedHeaderHeight]] to set the contents

TableRow#

• Optional TableRow: ComponentType<ItemProps & { context?: Context }>

Defined in src/interfaces.ts:161

Set to customize the item wrapping element. Default is tr.
